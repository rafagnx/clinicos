# ClinicOS

Sistema de Gestão de Clínicas.

## Stack
- **Frontend**: React (Vite) -> Hospedado na **Vercel**
- **Backend**: Node.js (Express) -> Hospedado no **Render**
- **Database**: PostgreSQL -> Hospedado no **Render**

## Como Atualizar o Site

O sistema agora usa **DEPLOY AUTOMÁTICO**.

1.  Faça suas alterações no código.
2.  Commit e Push para o GitHub:
    ```bash
    git add .
    git commit -m "Descrição do que você mudou"
    git push origin main
    ```
3.  Pronto!
    - O **Frontend** (Vercel) atualiza em ~1 minuto.
    - O **Backend** (Render) atualiza em ~3 minutos.
